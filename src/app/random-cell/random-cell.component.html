Description

Functions by using a pool of available cells that is kept track on and updated for each Cell draw through two Rule types "Injected" and "Restricted" explained below.


Injected:
Allows the Cell into the pool with a pre requirement that will be injected into the route if the Cell is drawn.

Example: Blue Rings Cell is available in the pool but will inject Purple Rings Cell into the route before itself if it hasn't been randomized in already.


Restricted:
Restricts the Cell from being available in the pool for the next Cell draw if the Rule still applies.

Example: Blue Rings Cell is not available in the pool before Purple Rings Cell has been randomized in.


This allows for scenarios where for example with "No FCS" enabled Snowy is available in the pool right after 20 cells are obtained, if Snowy is then drawn right after the program will map in the required End of Fire Canyon, End of Mountain Pass and two random cells as the next cells in the route, and will then add the Snowy cell that was randomized.

If this behaviour is unwanted and you for example want Snowy to be first available when it would be available by pure RNG (classic randomizer), turn the desired settings to "Restricted”.


If “No LTS” “No FCS” and “No Citadel Hops” are turned of this means you technically can get a Citadel Cell from the available "Cell Pool" right after Geyser Rock with the injected order -> End of Fire Canyon -> End of Mountain Pass -> Citadel Cell.
